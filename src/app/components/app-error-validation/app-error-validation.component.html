Error Header

The app-form-error-header component is used to generate a danger alert that informs the users how many errors exist on a form. You can also display a custom error message, which will override the default message. You may specify the following parameters:

errorHeaderMessage
A custom error message which should be displayed in the error header. If this parameter is not provided, the default message with the error count is displayed.
errorHeaderFormGroup
The FormGroup that will have its errors counted.
errorHeaderId
A custom id which we be applied to the error header. If this parameter is not provided, the id errorHeader is used.
