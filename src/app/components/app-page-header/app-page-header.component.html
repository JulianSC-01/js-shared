Page header component

Overview

The app-page-header component is used to generate a h1 header. For accessibility purposes, the h1 will automatically receive focus when the page is loaded. The browser's page title is also updated using the content of the h1 or the pageTitle parameter below. The text inside the header is output through ng-content. You may specify the following parameters:

pageTitle
This parameter specifies the text that will be set in the browser's title bar. If this parameter is not provided, the innerText of the h1 is used.

An example of the header can be found at the top of this page.
