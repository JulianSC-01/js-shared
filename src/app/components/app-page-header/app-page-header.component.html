Page header component

Overview

The app-page-header component is used to generate a h1 header. For accessibility purposes, the h1 will automatically receive focus when the page is loaded. The text inside the header is output through ng-content. You may specify the following parameters:

pageTitle
This is the text that will be set in the browser's title bar. If this parameter is not provided, the innerText of the h1 is used.

An example of the header can be found at the top of this page.
