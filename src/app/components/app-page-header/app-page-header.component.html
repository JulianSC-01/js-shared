Page header component

Overview

The app-page-header component is used to generate a h1 header. The text inside the header is output through ng-content. For accessibility purposes, the h1 will automatically receive focus when the page is loaded. You may specify the following parameters:

pageTitle (optional)
This parameter specifies the text that will be set in the browser's title bar. Defaults to the the innerText of the h1.

An example of the header can be found at the top of this page.
