Error Feedback

The app-form-error-feedback component is used to generate error messages that pertain to a specific form control. Typically, these message are displayed beneath the control. You may specify the following parameters:

formControl
This parameter specifies the AbstractControl that will be checked for errors.
errorFeedbackId (optional)
This parmaeter specifies the id attribute that will be applied to the error element, for accessibility purposes.
errorFeedbackMessages (optional)
This parameter specifies a key/value list that maps error codes to error messages. If any of the error codes are active on the control, the associated error message(s) will be displayed. Defaults to an empty list, which means no errors will ever be displayed.

Note: The app-form-error-feedback component is already contained within each app-form-input component.
