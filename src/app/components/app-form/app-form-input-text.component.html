Form Input - Text

The app-form-input-text component is used to generate a standard text input field which includes a label and error validation support. This component uses the ControlValueAccessor interface, so you should provide the form directive which identifies your control. You may specify the following parameters:

inputErrorMessageId (optional)
This parameter specifies the id attribute of the div element that will contain the error message(s), for accessibility purposes.
inputErrorMessages (optional)
This parameter specifies a key/value list that maps error codes to error messages. If any of the error codes are active on the control, the associated error message(s) will be displayed beneath the control. Defaults to an empty list, which omits any error feedback.
inputId (optional)
This parameter specifies the id attribute of the input element, for accessibility purposes.
inputLabelInvisible (optional)
This parameter specifies if the label element should be invisible, for accessibility purposes. Defaults to false.
inputLabelText (optional)
This parameter specifies the label's text. Defaults to undefined, which omits the label entirely.
inputMaxLength (optional)
This parameter specifies the maxLength attribute of the input element. Defaults to undefined, which omits the attribute.
inputPlaceholder (optional)
This parameter specifies the placeholder attribute of the input element. Defaults to undefined, which omits the attribute.
inputReadOnly (optional)
This parameter specifies if the input element should be read only. If provided, this must be true. Defaults to undefined, which omits the attribute.
inputRequired (optional)
This parameter specifies if the field should be marked as required, which adds the text (required) to the end of the label. Defaults to false.
inputSize (optional)
This parameter specifies the size attribute of the input element. Defaults to undefined, which omits the attribute.
inputType (optional)
This parameter specifies the type attribute of the input element. This must be either text or password. Defaults to text.
