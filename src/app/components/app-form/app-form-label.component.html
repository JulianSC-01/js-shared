Form Label

The app-form-label component is used to generate a label that is typically used with form input fields. The text inside the label is output through ng-content. You may specify the following parameters:

labelControlId (optional)
This parameter specifies the id attribute of the associated form control field, for accessibility purposes.
labelInvisible (optional)
This parameter specifies if the label should be invisible, for accessibility purposes. Defaults to false.
labelRequired (optional)
This parameter specifies if the label should be marked as required, which adds the text (required) to the end of the label. Defaults to false.

Note: The app-form-label component is already contained within each app-form-input component.
