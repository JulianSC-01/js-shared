Alert component

Overview

The app-alert component is used to generate a bootstrap style alert. The text inside the alert is output through ng-content. You may specify the following parameters:

alertId (optional)
This parameter specifies the id attribute of the alert.
alertRole (optional)
This parameter specifies the role attribute of the alert.
alertType (optional - Default: alert-info)
This parameter specifies the type of the alert, using one of the bootstrap CSS classes.

The alertType is specified as alert-info, or not provided.
