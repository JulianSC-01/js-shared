Page footer component

Overview

The app-page-footer component is used to generate a footer at the bottom of the page. You may specify the following parameters:

buildDate (optional)
This parameter contains a string or Date that represents the last time the application was built. If a Date is passed, the date is output based on the current locale.
versionNumber (optional)
This parameter contains a string or number that represents the version number.

An example of the footer can be found at the bottom of this page.
