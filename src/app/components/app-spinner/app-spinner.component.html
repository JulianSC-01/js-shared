Spinner component

Overview

The app-spinner component is used to generate an animated spinner icon. Additional HTML content can also be specified through ng-content. You may specify the following parameters:

spinnerAccessibleText
This parameter specifies the invisible text located within the spinner that is read by screen readers. If this parameter is not provided, the text "Loading..." is used.
spinnerColor
This parameter can be used to change the spinner's color. Simple text color classes from Bootstrap may be used. (Example: text-primary or text-secondary.) If this parameter is not provided, no custom color is applied.
spinnerSmall
This parameter can be set to true if you wish to use a small spinner.
spinnerStyle
This parameter specifies the style of spinner. It can be either spinner-border or spinner-grow. If this parameter is not provided, spinner-border is used.
