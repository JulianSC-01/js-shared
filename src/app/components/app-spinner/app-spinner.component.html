Spinner component

Overview

The app-spinner component is used to generate an animated spinner icon. Additional HTML content can also be specified through ng-content. You may specify the following parameters:

spinnerAccessibleText (optional)
This parameter specifies the invisible text located within the spinner, for accessibility purposes. Defaults to "Loading...".
spinnerColor (optional)
This parameter specifies the spinner's color, using text color classes from Bootstrap. (Example: text-primary or text-secondary.) Defaults to the standard text color.
spinnerSmall (optional)
This parameter specifies if you wish to use a small spinner. Defaults to false.
spinnerStyle (optional)
This parameter specifies the style of spinner. It can be either spinner-border or spinner-grow. Defaults to spinner-border.
